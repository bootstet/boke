# 1 Webpackзҡ„жһ„е»әжөҒзЁӢдё»иҰҒжңүе“ӘдәӣзҺҜиҠӮпјҹеҰӮжһңеҸҜд»ҘиҜ·е°ҪеҸҜд»ҘиҜҰе°Ҫзҡ„жҸҸиҝ°Webpackжү“еҢ…зҡ„ж•ҙдёӘиҝҮзЁӢгҖӮ

  • 1 еңЁйЎ№зӣ®дёӯе®үиЈ…webpackдҫқиө–пјҢnpm install webpack --dev
  • 2 еҲӣе»әwebpack.config.jsж–Үд»¶пјҢе№¶еҜ№е…¶иҝӣиЎҢзӣёе…ізҡ„й…ҚзҪ®

    mode: 'none',        // webpackе·ҘдҪңжЁЎејҸ

    output: {                 // иҫ“е…Ҙж–Үд»¶дҪҚзҪ®

            loader: 'url-loader',               // >- е°Ҹж–Үд»¶дҪҝз”ЁData UrlsпјҢеҮҸе°‘иҜ·жұӮж¬Ўж•°

# 2 Loaderе’ҢPluginжңүе“ӘдәӣдёҚеҗҢпјҹиҜ·жҸҸиҝ°дёҖдёӢејҖеҸ‘Loaderе’ҢPluginзҡ„жҖқи·ҜгҖӮ

  • LoaderжҳҜе®һзҺ°иө„жәҗжЁЎеқ—еҠ иҪҪпјҢе®һзҺ°ж•ҙдҪ“йЎ№зӣ®зҡ„жү“еҢ…
  • plugin жҳҜеўһејәWebpackиҮӘеҠЁеҢ–иғҪеҠӣпјҢи§ЈеҶійЎ№зӣ®дёӯиҮӘеҠЁеҢ–е·ҘдҪң
    • иҮӘеҠЁжё…йҷӨdistзӣ®еҪ•
    • жӢ·иҙқйқҷжҖҒweе№ҙиҮіиҫ“еҮәзӣ®еҪ•
    • еҺӢзј©жү“еҢ…иҫ“еҮәзҡ„д»Јз Ғ
  • ејҖеҸ‘loaderпјҡеҜјеҮәдёҖдёӘеҮҪж•°пјҢиҝҷдёӘеҮҪж•°дё»иҰҒеҒҡзҡ„дәӢжғ…е°ұжҳҜпјҢеҜ№иө„жәҗдёҖдёӘеӨ„зҗҶиҝҮзЁӢгҖӮиҝҷдёӘеҮҪж•°еңЁжӢҝеҲ°иө„жәҗеҠ иҪҪеҶ…е®№пјҲsourceпјүд»ҘеҗҺпјҢз„¶еҗҺз»ҸиҝҮиҝҷдёӘеҮҪж•°еҠ е·Ҙд»ҘеҗҺиҝ”еӣһеҖјиҫ“еҮәеҮәеҺ»гҖӮеҜјеҮәзҡ„ж–Үд»¶еҝ…йЎ»жҳҜjavascriptд»Јз ҒгҖӮдёҖдёӘзұ»дјјз®ЎйҒ“зҡ„еҺҹзҗҶгҖӮз„¶еҗҺеңЁconfigй…ҚзҪ®йҮҢrulesй…ҚзҪ®зӣёе…ізҡ„loaderгҖӮ
  • ејҖеҸ‘pluginжҖқи·Ҝпјҡ еңЁwebpackй’©еӯҗдёӯеҺ»жҢӮиҪҪзӣёеә”зҡ„д»»еҠЎпјҢжҸ’д»¶дёҖиҲ¬жҳҜдёҖдёӘеҮҪж•°пјҢжҲ–иҖ…жҳҜдёҖдёӘеҢ…еҗ«applyж–№жі•зҡ„еҜ№иұЎпјҢе®ҡд№үдёҖдёӘзұ»еһӢпјҢеңЁиҝҷдёӘзұ»еһӢе®ҡд№үдёҖдёӘapplyж–№жі•пјҢз„¶еҗҺеңЁдҪҝз”Ёж—¶е®ҡд№үдёҖдёӘиҝҷдёӘзұ»еһӢзҡ„е®һдҫӢеҺ»дҪҝз”ЁгҖӮжҸ’д»¶дёӯйҖҡиҝҮcompiler.hooksи®ҝй—®webpackдёӯзҡ„й’©еӯҗпјҢжүҫеҲ°зӣёеә”зҡ„й’©еӯҗеңЁзӣёеә”зҡ„ж–№жі•дёӯеҺ»еҒҡиҮӘе·ұзҡ„дәӢгҖӮ
